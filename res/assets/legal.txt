This app uses location API to suggest the relevant information in "there & back" mode. The app provides an option that the user can access any time to turn location API usage on or off. When this option is on, the app will store a single location selected by the user. This location will be used internally by the app and will not be sent to any services outside the app for any purposes, except for the cases described in Microsoft TouchDevelop Privacy Statement (see below). The app provides a screen that the user can access any time, which displays the location the app has stored.
